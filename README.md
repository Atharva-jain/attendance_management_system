# 🎓 Attendo - AI-Based Attendance Management Android App[cite: 2]

Imagine a world where taking attendance is seamless, secure, and smart[cite: 2]. Attendo transforms how classrooms operate by integrating cutting-edge deep learning technology for face recognition and location verification. 

Attendo is a high-security attendance system that uses a 3-step verification process—Location, Facial Recognition, and Class Codes—to eliminate attendance fraud and automate classroom management.

## 🛠️ Technology Stack
The architecture of Attendo is designed for real-time performance and high security[cite: 2].

### Core Infrastructure
*   **Frontend:** **XML** for building responsive and intuitive user interfaces[cite: 2].
*   **Backend:** **Kotlin**, providing a modern, safe, and efficient environment for Android logic[cite: 2].
*   **Database:** **Firebase**, used for real-time data synchronization of attendance records and secure user authentication[cite: 2].
*   **Artificial Intelligence:** **Python**, powering the deep learning models used for facial recognition and verification[cite: 2].

### Verification Logic
To ensure accuracy, the app utilizes a unique 3-step protocol:
1.  **Geofencing:** Verifies the student is within the teacher's physical radius using GPS[cite: 2].
2.  **Biometrics:** Uses AI-driven face verification to ensure the correct student is present[cite: 2].
3.  **Dynamic Codes:** Requires a secret class code generated by the teacher for final confirmation[cite: 2].

## ⚙️ Functionality

### 👨‍🏫 For Teachers
*   **Virtual Classrooms:** Set up classrooms by college, semester, year, and branch[cite: 2].
*   **Live Class Control:** Specify class names, current location, and unique access codes[cite: 2].
*   **Administrative Oversight:** Full control to add or remove students and mark manual attendance if necessary[cite: 2].
*   **Data-Driven Insights:** View individual attendance percentages and total class presence instantly[cite: 2].

### 👨‍🎓 For Students
*   **Secure Enrollment:** Sign up with profile details and an initial face scan for future recognition[cite: 2].
*   **Attendance Tracking:** Review records for past classes and monitor your overall attendance percentage[cite: 2].
*   **Seamless Check-in:** Join live classes and complete the 3-step verification in seconds[cite: 2].
